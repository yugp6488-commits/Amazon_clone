# 🛒 Amazon Clone - Full Homepage UI

This is a high-fidelity frontend clone of the **Amazon.in** homepage. The goal of this project was to master complex CSS layouts, specifically focusing on the interaction between **Flexbox** and **CSS Grid** to create a responsive, production-grade user interface.

## 🚀 Live Preview
Check out the Live Site   : 
(https://yugp6488-commits.github.io/Amazon_clone/)


## 🛠️ Technical Highlights
* **Advanced Navigation Bar:** Replicated the multi-layered navbar featuring the search dropdown, country-specific flags, and the "Account & Lists" hover menus.
* **Hero Section & Overlapping Cards:** Used z-index and negative margins to allow the product category cards to "float" over the hero image, mimicking Amazon's distinct layout.
* **Responsive Grid System:** Implemented a flexible grid for product listings that adjusts based on screen width.
* **Custom Iconography:** Integrated via **Font Awesome 7.0.1** for the location, search, and shopping cart elements.

## 📂 Project Structure
* **index.html**: Semantic HTML5 structure for better accessibility.
* **index.css**: Custom CSS3 styling (No external frameworks like Bootstrap used).
* **assets/**: Optimized images for the hero slider and product categories.

## 💻 Tech Stack
* **HTML5**: For the structural backbone.
* **CSS3**: For custom animations, Flexbox, and Grid layouts.
* **Font Awesome**: Scalable vector icons.
